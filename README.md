Splice Laravel API Project

Steps for Intsalltion :-

1) git clone https://github.com/DevanshiDarji/Splice-Laravel-Project.git
2) cd Splice-Laravel-Project
3) composer install
4) php artisan migrate --seed
5) php artisan serve
